## 关于项目管理 

Coding.net 在 Git 代码托管的基础上自主研发一套供软件开发人员使用的项目管理和团队协作工具。为了提高团队协作下软件开发的效率。

团队协作以项目为单位。一个账号可以创建多个项目，一个项目可以添加多个项目成员。

 主要功能如下：
 
 - Git 代码操作整合
 - 任务管理
 - 讨论管理
 - 在线文件管理
