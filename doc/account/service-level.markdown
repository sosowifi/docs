Coding 针对不同的资源需求，提供给用户不同的服务。

### 项目升级是什么

项目升级是Coding平台推出为项目提供更多资源更高服务的一项收费业务模块，现阶段推出的付费种类有开发版和高级版这两个版本。

### 项目升级详情和收费
