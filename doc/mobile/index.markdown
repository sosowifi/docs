## 移动端 App 

Coding app 是基于 coding.net 网站设计的一款便捷的软件开发协作工具。可以用它来随时随地的查看 https://Coding.net 上的精彩项目、任务动态、项目文档、项目代码、冒泡社交。

## 主要功能 

- 任务管理
- 项目文档
- 项目讨论
- 代码托管
- 冒泡社交
